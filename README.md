# Arkanoid (test task)

Клон Arkanoid под веб в стиле NES/Famicom.
Сделан на чистом `HTML + CSS + JS`, без фреймворков и без сборки.

## Как запустить

Обычным двойным кликом по `index.html` игра не стартует, потому что код разбит на модули (`import/export`) и браузер блокирует их из `file://`.

Самый простой вариант:

```bash
node server.mjs
```

Или просто запустить `start.bat`.

После старта открыть:

```text
http://127.0.0.1:8080
```

В VS Code можно нажать `F5` (`Arkanoid JS (Run)`), там выполняется та же команда `node server.mjs`.

## Что в игре уже есть

- меню: `HOW TO PLAY`, `1 PLAYER GAME`, `2 PLAYERS GAME` (disabled), `LEADERBOARD`
- полноценный игровой цикл, экраны переходов, пауза с выбором
- 35 уровней и босс DOH
- капсулы `E/S/C/L/D/B/P`, серебряные и неразрушаемые блоки
- враги, вылетающие из правого шлюза
- таблица рекордов Top 5 и сохранение в `localStorage`
- мини-туториал

## Управление

- `A/D` или `Left/Right` - движение платформы
- `Space` / `Enter` / ЛКМ - старт, запуск мяча, подтверждение
- `X` - выстрел (если активирован лазер)
- `P` или `Esc` - пауза
- в меню: `W/S` или `Up/Down`, подтверждение `Space/Enter`
- в паузе: `A/D` или `Left/Right`, подтверждение `Enter`
- ускорение времени: ЛКМ = `x2`, ЛКМ + ПКМ = `x4`

## Структура

- `index.html`, `styles.css`, `game.js` - точка входа
- `src/` - логика игры, разбитая по модулям
